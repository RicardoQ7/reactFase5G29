RESPONDE LAS SIGUIENTES PREGUNTAS

1. ¿Que es el Box Model?

a. Es una especie de caja, donde se agregan los tipos de estilos para la página web
b. En los navegadores, todo se construye a partir de cajas. Debemos entender, es que cada elemento que definimos en un documento HTML se mostrará en el navegador como una caja rectangular
c. Es una interfaz que dos sistemas de computación utilizan para intercambiar información de manera segura a través de Interne

2. ¿Cuales son las cuatro principales propiedades del Box Model'

a. Alto, ancho, área y base.
b. left, right, width y height.
c. Contenido, borde, padding y el margin.

3. La definición de contenido en un box model es:

a. Se trata del contenido HTML del elemento (las palabras de un párrafo, una imagen, el texto de una lista de elementos, etc.)
b. línea que encierra completamente el contenido y su relleno.
c. Separación opcional existente entre la caja y el resto de cajas adyacentes.

4. En el modelo de caja en CSS existen otras dos propiedades para modificar el borde, cuales son:

a. Border-radius: permite redondear las esquinas del borde cambiando su radio.
b. Border-image: permite establecer una imagen como borde.
c. a y b son correctas

5. ¿A que hacen referencia el width y el height?

a. Aplica exactamente la misma lógica que con el padding (sustituyendo la palabra padding por margin y todo el resto igual).
b. A la altura y al ancho de una caja html, Los elementos en línea no hacen caso del width ni el height que hayas podido indicar en la hoja de estilos. Para que lo hagan hay que transformarlos a elementos bloque o a elementos inline-block.
c. Los elementos bloque si respetan todos los márgenes que hayamos podido indicar. Pero atención, los márgenes inferior y superior de dos cajas una encima de la otra se fusionan.
